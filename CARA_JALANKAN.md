# Panduan Cepat Menjalankan Portofolio

Ikuti langkah-langkah di bawah ini agar semua proyek Anda (PHP & Node.js) dapat berjalan dengan lancar.

## 1. 🚀 Menjalankan E-Commerce (NovaMart)
Karena NovaMart menggunakan teknologi modern (Next.js), ia membutuhkan server khusus.
- Buka folder `e:/portofolio randi sahputra/`.
- Klik dua kali file **`JALANKAN_NOVAMART.bat`**.
- Tunggu sampai muncul tulisan **"Ready"** atau **"Started"**. 
- Sekarang Anda bisa mengklik tombol di portofolio untuk membukanya.

## 2. 🤖 Menjalankan AI Chatbot
- Masuk ke folder `02-ai-chatbot`.
- Buka Terminal/CMD di sana.
- Ketik: `npm start`.

## 3. 🌐 Menjalankan Proyek PHP (Qorry, SPK, Tara)
- Masuk ke folder masing-masing.
- Pastikan folder proyek sudah Anda letakkan di dalam folder **`htdocs`** XAMPP.
- Klik tombol di portofolio, dan pastikan XAMPP (Apache & MySQL) sudah dalam posisi **START**.

---

## 🛠️ Tips Jika Error "Site can't be reached"
1. **Port Terpakai**: Pastikan tidak ada aplikasi lain (seperti Skype atau server lain) yang menggunakan port 3000.
2. **Terminal Ditutup**: Jika Anda menutup jendela hitam (CMD) setelah menjalankan proyek, maka proyek tersebut akan otomatis mati. Biarkan jendela tersebut tetap terbuka saat Anda melakukan demo.
